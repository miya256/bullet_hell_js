# 弾幕シューティング

▶ **Play:** https://miya256.github.io/bullet_hell_js/

## 概要
ブラウザで遊べるシューティングゲームです。  
※ 未完成です

## 操作方法

### タイトル画面
**プレイする**ボタンを押しましょう。

### マップ画面
プレイボタンを押すと、マップ画面になります（青が海、緑が島のつもりです）。  
背景を動かせますが（プレイヤーが映ってないけど中央にプレイヤーがいる想定）、今は特に意味はないです。  
Enter を押すとゲームが始まります。

### プレイ中
- ← → ↑ ↓：移動
- Z：ショット
- Shift：低速移動（押している間はため状態になります。離すと発動します）

## ルール
- 敵の弾をよけましょう
- 敵に自機の弾を当てましょう
- HP が 0 になるとゲームオーバーです
